# Strong-Pass
StrongPass is a secure password manager and password security analyzer built with JavaScript, Node.js, Firebase Authentication, and Firestore. It features AES-GCM encryption, PBKDF2 key derivation, password strength analysis with zxcvbn, a secure password generator, and a zero-knowledge approach to protect user credentials.
